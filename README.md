# Web Playground

## Description  
This repository contains the deliverables for Day 4 and Day 5. The deliverables include HTML, CSS, JavaScript files, and related documentation.

## Repository Structure  
- Deliverables are organized in separate folders by day.
- Teh **Day-4** folder contains deliverables for Day-4
- The **Day 5** folder contains the profile page code.

## Running the Project  
Open your files in VS Code and start Live Server or open the HTML files directly in your browser.

## Deployment
