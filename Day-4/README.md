## Development Environment Setup

### VS Code Settings  
The repository includes a committed `.vscode/settings.json` file configuring Prettier as the default formatter and enabling formatting on save for consistent code style.

### Installed Extensions  
- **Prettier - Code Formatter:** Enforces consistent code formatting.  
- **ESLint:** JavaScript linting with auto-fix capabilities on save.  
- **Code Runner:** Quickly run snippets in the integrated terminal.  
- **Live Server:** Launches a development server with live reload feature.  
- **Better Comments:** Improves code comments highlighting for better readability.  
- **GitHub Copilot:** AI-based code completion assistant for supported languages.  
- **Java Extension Pack:** Java SDK support with Maven and Gradle configured for project builds.  
- **Spell Checker (cSpell):** Code spell checker with suggestions and diagnostics.  
- **HTML Language Features:** Provides syntax highlighting and formatting support for HTML files.
